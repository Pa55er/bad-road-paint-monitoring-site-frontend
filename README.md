<h2>About Project</h2>
<h4>[불량 도로 페인트 모니터링 사이트]</h4>

차량에 탑재될 블랙박스 같은 카메라 장치에서 딥러닝 모델을 이용하여 도로주행시 불량 도로 페인트들을 찾아내 백엔드 서버로 해당 도로의 이미지와 위치 정보를 백엔드 서버에 전달합니다. 국토교통부와 같은 도로 유지 보수 기관에 불량 도로 페인트들의 정보를 제공하고 관리할 수 있도록 모니터링 사이트를 구축했습니다.
